**瘦身打卡管理系统**

**项目介绍**

本项目使用HTML和CSS开发，是一个瘦身打卡管理静态页面。

&#x20;**文件结构**

&#x20;├── index.html

&#x20;├── styles/

&#x20;│   └── main.css

&#x20;├── assets/

&#x20;└── README.md

**页面功能**

&#x20;1. 顶部区域：页面标题、搜索框、新增记录按钮

&#x20;2. 左侧导航：统计总览、打卡记录、个人设置

&#x20;3. 数据卡片：今日待完成、今日已完成

&#x20;4. 近期瘦身记录列表：4条记录，包含名称、日期、状态

&#x20;5. 附加模块：今日瘦身小贴士

&#x20;**使用方法**

&#x20;双击index.html文件，即可在浏览器中打开查看页面效果。
